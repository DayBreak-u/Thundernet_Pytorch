## Requirements
* pytorch 1.2.0
* torchvision 0.4



## Lib Prepare 
```sh
git clone https://github.com/ouyanghuiyu/Thundernet_Pytorch.git
```

### Build  
```sh
cd lib && python setup.py  build_ext --inplace
cd psroialign/PSROIAlign && sh build.sh 
 ```   
## Data Prepare 
Download VOC0712 datasets 
ln -s "YOUR PATH" data

## Train
```sh
cd script
sh  train_49.sh
sh  train_146.sh
sh  train_535.sh
```
